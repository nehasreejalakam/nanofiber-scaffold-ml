📌 Overview

This project focuses on developing an intelligent system for predicting wound healing performance and recommending optimal nanofibrous scaffold designs using machine learning and biomedical knowledge.

It integrates data-driven modeling, literature-based scoring, and clinical decision rules to evaluate scaffold properties and optimize their suitability for different wound types.

🎯 Objectives
Predict healing time using machine learning models
Recommend optimal scaffold properties (inverse design)
Evaluate scaffold performance using scientific benchmarks
Apply clinical rules for different wound conditions
Improve scaffold design for effective tissue regeneration

🧪 Features
Machine learning-based healing prediction
Inverse scaffold design recommendation
Dataset-specific analysis using real parameters
Literature-based scoring (cell viability, structure, biocompatibility, etc.)
Clinical decision rules (diabetic, chronic, bone, surgical wounds)
Bootstrap validation and reliability assessment
Interactive visualization using widgets and plots

🛠️ Technologies Used
Python
NumPy, Pandas
Scikit-learn
Matplotlib, Seaborn
SciPy
ipywidgets

📊 Dataset

Utilized a structured dataset of nanofibrous scaffold properties including cell density, fiber diameter, pore size, contact angle, mechanical strength, degradation time, and surface roughness.
Integrated dataset-specific parameter mapping and normalization to enable accurate model training, analysis, and scaffold performance prediction.

📊 Workflow
Data preprocessing and feature alignment
Model prediction of healing performance
Scaffold parameter optimization
Scoring using literature and clinical rules
Validation using bootstrap and statistical analysis
Visualization and recommendation generation

📈 Results

The system provides:

Predicted healing time
Performance classification
Ranked scaffold recommendations
Reliability and uncertainty analysis
